  ## Current NSE model is stale

  The fixed ₹0.05 tick is no longer generally correct. NSE cash-market ticks now vary from ₹0.01 to ₹5 according to price bands and are reviewed monthly. The project quantizes all incoming
  OHLC prices using one fixed tick, which can create impossible entries, stops and fills. See the official NSE tick-size circular
  (https://nsearchives.nseindia.com/content/circulars/CMTR67133.pdf).

  The configured transaction rate is also stale. From March 2026, NSE’s cash-market transaction charge is ₹306.99 per crore on each side, with IPFT in addition. The model omits or
  incompletely models IPFT, SEBI fees, GST and buyer-side stamp duty. See the NSE transaction-charge circular (https://nsearchives.nseindia.com/content/circulars/FA73061.pdf) and official
  levy schedule (https://www.nseindia.com/static/invest/first-time-investor-sebi-turnover-fees-stt-other-levies).

  Expiry handling remains overbroad. Individual stock derivatives expire on the last Tuesday, shifted to the previous trading day for a holiday; the code simply tests whether every date is
  Tuesday. See NSE contract specifications (https://www.nseindia.com/static/products-services/equity-derivatives-contract-specifications).
